# house-release
发布代码house

## 项目开发前操作
### 安装nodejs
安装nodejs,推荐用`nvm`或`n`进行node的版本管理

### 安装yarn替代npm
官网: [https://yarnpkg.com/zh-Hans/](https://yarnpkg.com/zh-Hans/)

## 项目安装与运行

```
   
    yarn #或yarn install
    
    npm run build
    npm run dev
    
浏览器打开 http://localhost:8080/ 进入 build/pages/ 下相应的目录
```
